Какую проблему решает бот: сбор обратной связи и передача её руководителю
Почему выбрали именно эту задачу: мне, как HR-у, нужен инструмент для сбора обратной связи для сотрудников

Промпт для LLM

Представь, что ты лучший разработчик Telegram-ботов на Python

Твоя задача: создать Telegram-бота на Python с использованием библиотеки python-telegram-bot

Функционал бота:
- Собирает мнения команды по вопросам: через кнопку обратной связи — сбор данных в базу
Формирует сводные отчеты
Отправляет уведомления руководителю — сообщение перессылается вместе с username и telegram_id, кто присылает, админу с telegram_id 975902763 или @sevbarv
Ведет статистику — количество обратной связи. Через кнопку "Обратная связь"
- Команды: кнопки "Обратная связь", "Статистика" и "Формирование отчета" в формате XLSX
- Какие должны быть ответы: приветственное слово при запуске после нажатия кнопки "Старт" в свободной форме
- Хранить данные: username, telegram_id, обратная связь (feedback)

Требования:
- Бот должен быть простым и понятным, то есть хорошо и чисто написан для проверки экспертом
- Код должен быть хорошо прокомментирован, так, чтобы студент из другой сферы (не IT), понял, что где находится
- Использовать файл для хранения данных SQLite
- Добавить обработку ошибок, чтобы можно было спокойно показать преподавателю

Создай:

1. Файл bot.py с кодом бота
2. Файл requirements.txt с зависимостями
3. Файл README.md с инструкцией по запуску
4. Файл .env.example для примера конфигурации
